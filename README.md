## Konfigurimi

Se pari ju duhet te importoni databazen e Web-it ne **SQL Server** e cila gjendet ne folderin Databaza me emrin **FinanCareDB.sql** ose **FinanCareDBEmpty.sql**, pastaj duhet te behet konfigurimi i Connection String ne WebAPI/WebAPI/appsettings.json dhe duhet te nderrohet emri i Server me ate te serverit tuaj.

Pasi qe te behet konfigurimi ju mund te kyqeni me keto te dhena:

| **Email** | **Password** | **Aksesi**                    |
| ------------ | ------------ | ----------------------------- |
| admin@techstore.com        | Admin1@        | Administrator (Akses i Plote) |
| menagjer@techstore.com       | Menagjer1@     | Menagjer (Akses i Pjesshem)   |
| user@techstore.com           | User1@         | Klient (Akses i thjesht)      |
